# Change Order 02: Expanded Security Operations Scope

Effective date: October 10, 2025

## Reason

Following the Q3 governance review, Ridgeway requested expanded monitoring around privileged access alerts, phishing investigations, and externally exposed systems.

## Scope Added

- daily review of privileged-access exceptions
- phishing triage runbook updates
- monthly external attack-surface validation
- executive security appendix added to governance package

## Delivery Impact

- one security analyst reassigned to the Ridgeway account
- monthly governance packet now includes vulnerability aging and control exceptions
- Q4 penetration testing results must be reviewed with client leadership
